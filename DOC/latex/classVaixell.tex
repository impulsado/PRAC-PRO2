\doxysubsection{Descripción detallada}
Representa un vaixell amb una determinada quantitat de productes per comprar i vendre. 

Un objecte de la classe \mbox{\hyperlink{classVaixell}{Vaixell}} conté una parella de productes per comprar i vendre, així com un registre de les últimes ciutats visitades.
